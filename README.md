# Artistry by Sara MUA – Frontend (Angular)

Este repositorio contiene el frontend de la plataforma Artistry by Sara MUA.

# Tecnologías principales

- Angular 17+

- TypeScript

- NgRx

- RxJS

- Angular Router

- Angular Forms (Reactive Forms)

- HttpClient + Interceptor JWT

# Instalación

Clonar el repositorio:

        git clone https://github.com/monicahdev/artistry_app.git

        cd artistry_app

# Instalar dependencias:

        npm install

# Ejecutar en modo desarrollo:

        ng serve

Abrir en navegador:

        http://localhost:4200

# Configuración

La URL del backend (FastAPI) se define en el fichero:

**src/environments/environment.ts**

Ejemplo:

export const environment = {
production: false,
apiUrl: 'http://127.0.0.1:8000'
};

# Autenticación

El frontend utiliza tokens JWT. El token se almacena en localStorage. Se añade automáticamente a cada petición mediante AuthInterceptor. Los guards controlan acceso a rutas protegidas.

# Estructura principal del proyecto

src/app/
├── admin/ # Panel de administración
├── auth/ # Login, registro, NgRx de autenticación
├── bookings/ # Reservas de servicios de maquillaje
├── makeup-services/ # Catálogo público de servicios
├── online_classes/ # Clases online disponibles para el usuario
├── user/ # Datos del usuario autenticado
├── shared/ # Header, footer, guards, interceptor, servicios comunes
└── public/ # Home, About Me, Portfolio

# Estado global (NgRx)

La aplicación utiliza NgRx para gestionar:

- Autenticación (auth.state)

- Usuario (user.state)

- Servicios de maquillaje

- Reservas

- Clases online

Esto permite un flujo de datos predecible, escalable y fácil de depurar mediante Store DevTools.
